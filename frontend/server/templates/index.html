Welcome to VM Tool Server

Use the options below to access different VM tool functions:

📋 List Files

List all files in a VM disk image with detailed metadata including size, permissions, and modification time

📄 Files JSON

Get file listings as JSON format with numbered entries for easy programmatic access

💾 Disk Meta Data

View comprehensive disk metadata including file counts, total sizes, and per-user/group breakdowns

📖 File Contents

Read and display the contents of any file inside a VM disk image

🔢 Contents Format

Read file contents in specific formats (hex or binary) for low-level analysis

🔍 Check File Exists

Verify if a file exists in the guest image and get detailed file information

⚖️ File Compare

Compare the contents of two files side-by-side from different VM disk images

🔄 Difference in Files

Compare complete file lists between two VM disk images and identify unique and common files

📁 Compare Files In Directory

Compare files within specific directories across two VM disk images with detailed diff view

🔬 Blocks Comparison

Compare two VM disk images block by block and identify differing blocks with detailed metadata

📦 Block Data Viewer

View raw block data from a VM disk image in hex or binary format for low-level analysis

🧩 Block Contents Compare

Enter two images, a block number and size, choose hex/bits, and view the block contents side by side

🔁 Convert Disk Image

Convert between qcow2, vdi, vmdk, and raw formats using qemu-img with a simple UI.

🚀 Run QEMU VM

Boot a disk image with QEMU/KVM. Optional UEFI and convert .vdi/.vmdk to .qcow2 before running.

📦 Run VirtualBox VM

Launch a VM from .vdi/.vmdk (or convert .qcow2 to .vdi) with CPU/RAM/VRAM and bridged/NAT options.

💼 Run VMware VM

Run a .vmdk (or convert .vdi/.qcow2 to .vmdk) and start via vmrun/vmplayer, with NIC and nogui options.
